Git

  • git clone - Клонирует удаленный репозиторий.
  • git status - Показывает все изменения файлов и состояние репозитория.
  • git add - Добавляет изменения файлов в индекс.
  • git commit - Сохраняет изменения.
  • git push - Отправляет изменения на удаленный репозиторий.
  • git log - Выводит список всех коммитов.
  • git pull - Получает изменения из удаленного репозитория в локальный.
